最大k乘积问题

Aug 25, 2022
阅读时长: 2 分钟

问题描述

设I是一个n位十进制整数。如果将I划分为k段，则可得到k个整数。
这k个整数的乘积称为I的一个k乘积。

编程任务

试设计一个算法，对于给定的I和k，求出I的最大k乘积。

解题思路

证明该问题满足最优子结构

假设数I的最大k乘积为f(I, k)，其对应划分方法的最后一段其值为α，前k-1段的为β，乘积为g(β, k-1)，f(I, k) = α × g(β, k-1)。

则要求，g(β, k-1)为β的最大k-1乘积。

否则，必然存在h(β, k-1) > g(β, k-1) 使得 α × h(β, k-1) > α × g(β, k-1)既α × h(β, k-1) > f(I, k)。

∴ f(I, k)不是I的最大k乘积，与假设违背。

综上所述：最大k乘积满足最优子结构

#include <stdio.h>
#include <stdlib.h> 

int n;
int k;
int num[10];
int dp[10][10];
int m[10][10];

void init()  
{  
    for (int i = 1; i <= n; i++)
    {
        m[i][i] = num[i];  
    }
    for (int i = 1; i <= n; i++)
    {
        for (int j = i+1; j <= n; j++)
        {  
            m[i][j] = m[i][j-1]*10 + m[j][j];  
        }
    }
}//初始化

int main()
{
    scanf("%d", &n);
    scanf("%d", &k);
    for (int i=1; i <= n; i++)
    {
        char ch;
        scanf("%s", &ch);
        num[i] = ch-'0';
    }
    
    init();

    for (int i = 1; i <= n; i++)
    {
        for(int j = 0; j < i; j++)
        {
            if(j == 0)
            {
                dp[i][j] = m[1][i];
                continue;
            }
            for(int k = 1; k < i; k++)
            {
                if (dp[i][j] < dp[k][j-1]*m[k+1][i])
                {
                    dp[i][j] = dp[k][j-1]*m[k+1][i];
                }
            }
        }
    }
    printf("%d\n", dp[n][k-1]);
    system("pause");
    return 0;  
}
